# PROD_CREATE_POS_001 

## Название
Успешное создание товара (admin/seller)

## Связанные требования
1. REQ_PRODUCT_002 - Создание товара (admin/seller)

## Предусловия
1. В системе существуют пользователи:
   - tadmin (роль admin)
   - tseller (роль seller)
2. Пользователь авторизован под логином tadmin или tseller и находится в разделе "Товары"

## Тестовые данные
1. Наименование товара: строка с названием (буквы, цифры или спецсимволы, например, Test product)
2. Цена: целое положительное число ≥ 1 (например, 100)
3. Наличие товара: валидное значение (например, чекбокс "В наличии" установлен)

## Шаги воспроизведения
1. Нажать кнопку "Новый товар"
2. Заполнить поля формы данными из тестовых данных
3. Нажать кнопку "Сохранить"

## Ожидаемый результат
1. Товар успешно создаётся
2. Пользователь остаётся авторизованным
3. Новый товар отображается в списке товаров с указанными наименованием, ценой и выбранным признаком наличия

## Статус
Active

## Приоритет
High

## Окружение
- ОС: macOS 26.2
- Браузер: Comet
- URL: http://localhost:3001

## Дополнительно
Тест должен быть выполнен дважды:
 - под пользователем tadmin (admin)
 - под пользователем tseller (seller)
