# AUTH_REG_POS_001 

## Название
Успешная регистрация пользователя

## Связанные требования
1. REQ_AUTH_001 - Требования к логину
2. REQ_AUTH_002 - Требования к паролю

## Предусловия
1. Пользователь не зарегистрирован в системе 
2. Пользователь не авторизован

## Тестовые данные
1. Логин: строка из латинских букв в нижнем регистре, длиной от 1 символа (например, tuser)
2. Пароль: строка длиной от 6 символов, содержащая хотя бы одну заглавную букву, одну строчную букву, одну цифру и один спецсимвол (например, User123!)

## Шаги воспроизведения
1. Открыть страницу "Вход в панель" http://localhost:3001/login
2. Ввести логин из тестовых данных
3. Ввести пароль из тестовых данных
4. Нажать кнопку "Зарегистрироваться"

## Ожидаемый результат
1. Пользователь успешно зарегистрирован
2. Происходит редирект в раздел "Товары"
3. Отображается сообщение: "Успех. Зарегистрировались и авторизовались под: tuser"

## Статус
Active

## Приоритет
High

## Окружение
- ОС: macOS 26.2
- Браузер: Comet
- URL: http://localhost:3001
