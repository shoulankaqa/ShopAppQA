# AUTH_REG_NEG_001 

## Название
Регистрация пользователя с уже существующим логином

## Связанные требования
1. REQ_AUTH_001 - Требования к регистрации

## Предусловия
1. Пользователь t_user зарегистрирован в системе 
2. Пользователь t_user не авторизован

## Тестовые данные
1. Логин: `t_user` 
2. Пароль: `User123!` 

## Шаги воспроизведения
1. Открыть страницу "Вход в панель" http://localhost:3001/login
2. Ввести логин из тестовых данных
3. Ввести пароль из тестовых данных
4. Нажать кнопку "Зарегистрироваться"

## Ожидаемый результат
1. Регистрация не выполняется
2. Пользователь не создается 
3. Появляется сообщение: "Пользователь t_user уже существует"

## Статус
Active

## Приоритет
High

## Окружение
- Shop App: 1.0.0
- ОС: macOS 26.2
- Браузер: Comet (сборка на основе Perplexity)
  - Версия Comet: 148.0.7778.257 (официальная сборка)
  - Номер сборки Perplexity: 1743
- URL: http://localhost:3001
