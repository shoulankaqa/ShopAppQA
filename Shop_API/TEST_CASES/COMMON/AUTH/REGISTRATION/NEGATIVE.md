# AUTH_REG_NEG_001 

## Название
Регистрация с уже существующим логином

## Связанные требования
1. REQ_AUTH_001 - Требования к логину
2. REQ_AUTH_002 - Требования к паролю

## Предусловия
1. Пользователь с логином tuser зарегистрирован в системе
2. Пользователь не авторизован

## Тестовые данные
1. Логин: tuser
2. Пароль: строка длиной от 6 символов, содержащая хотя бы одну заглавную букву, одну строчную букву, одну цифру и один спецсимвол (например, Pass1!)

## Шаги воспроизведения
1. Открыть страницу "Вход в панель" http://localhost:3001/login
2. Ввести логин из тестовых данных
3. Ввести пароль из тестовых данных
4. Нажать кнопку "Зарегистрироваться"

## Ожидаемый результат
1. Регистрация не выполняется
2. Пользователь не создается 
3. Появляется сообщение: "Пользователь tuser уже существует"

## Статус
Active

## Приоритет
High

## Окружение
- ОС: macOS 26.2
- Браузер: Comet
- URL: http://localhost:3001
