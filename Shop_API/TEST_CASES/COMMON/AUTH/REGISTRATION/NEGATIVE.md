# AUTH_REG_NEG_001 

## Название
Регистрация пользователя с уже существующим логином

## Связанные требования
1. REQ_AUTH_001 - Требования к регистрации

## Предусловия
1. Пользователь tuser зарегистрирован в системе 
2. Пользователь tuser не авторизован

## Тестовые данные
1. Логин: tuser 
2. Пароль: User123! 

## Шаги воспроизведения
1. Открыть страницу "Вход в панель" http://localhost:3001/login
2. Ввести логин из тестовых данных
3. Ввести пароль из тестовых данных
4. Нажать кнопку "Зарегистрироваться"

## Ожидаемый результат
1. Регистрация не выполняется
2. Пользователь не создается 
3. Появляется сообщение: "Пользователь tuser уже существует"

## Статус
Active

## Приоритет
High

## Окружение
- ОС: macOS 26.2
- Браузер: Comet
- URL: http://localhost:3001

# AUTH_REG_NEG_002

## Название
Регистрация пользователя с пустым полем "Логин"

## Связанные требования
1. REQ_AUTH_001 - Требования к регистрации

## Предусловия
1. Пользователь не зарегистрирован в системе
2. Пользователь не авторизован

## Тестовые данные
1. Пароль: строка длиной от 6 символов, содержащая хотя бы одну заглавную букву, строчную, цифру и спецсимвол (например: NoLogin123!)

## Шаги воспроизведения
1. Открыть страницу "Вход в панель" http://localhost:3001/login
2. Оставить поле "Логин" пустым
3. Ввести пароль из тестовых данных
4. Нажать кнопку "Зарегистрироваться"

## Ожидаемый результат
1. Регистрация не выполняется
2. Пользователь не создается 
3. Появляется сообщение: "Требуется имя пользователя и пароль"

## Статус

## Приоритет
High

## Окружение
- ОС: macOS 26.2
- Браузер: Comet
- URL: http://localhost:3001
