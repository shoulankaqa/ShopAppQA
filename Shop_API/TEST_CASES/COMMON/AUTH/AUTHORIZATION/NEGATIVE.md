# AUTH_LOGIN_NEG_001 

## Название
Авторизация с несуществующими учетными данными

## Связанные требования
1. REQ_AUTH_004 - требования к авторизации

## Предусловия
1. Пользователь tuser отсутствует в системе (при необходимости удалить пользователя, если он существует)

## Тестовые данные
1. Логин: tuser
2. Пароль: Pass1

## Шаги воспроизведения
1. Открыть страницу "Вход в панель" http://localhost:3001/login
2. Ввести логин из тестовых данных
3. Ввести пароль из тестовых данных
4. Нажать кнопку "Войти"

## Ожидаемый результат
1. Авторизация не выполняется
2. Пользователь остается на странице "Вход в панель"
3. Отображается сообщение: "Неверные учетные данные"
