# PROD_CART_POS_001

## Название
Успешное добавление товара в корзину 

## Связанные требования
1. REQ_CART_001 — Добавление товара в корзину

## Предусловия
1. В системе существует пользователь t_user 
2. Пользователь t_user авторизован и находится в разделе "Товары"
3. В системе существует хотя бы один товар со статусом "В наличии"
4. Корзина пользователя t_user пуста

## Шаги воспроизведения
1. Нажать круглую кнопку с иконкой корзины напротив товара со статусом "В наличии" один раз
2. Нажать на иконку корзины в интерфейсе для просмотра содержимого корзины

## Ожидаемый результат
1. В корзине появился выбранный товар
2. У товара в корзине отображается количество = 1
3. На круглой кнопке с иконкой корзины напротив того же товара отображается количество выбранного товара (1)
4. На иконке корзины отображается количество добавленных товаров (1)

# PROD_CART_POS_005

1. REQ_CART_003 — Изменение количества товара

1. В системе существует пользователь t_user
2. Пользователь t_user авторизован и находится в разделе "Товары"

1. На странице "Товары" нажать круглую кнопку с иконкой корзины напротив того же товара ещё один раз
2. Нажать на иконку корзины в интерфейсе для просмотра содержимого корзины
3. Проверить, что количество этого товара в корзине увеличилось (было 1, стало 2)
4. Проверить, что на круглой кнопке с иконкой корзины напротив этого товара на странице "Товары" также отображается актуальное количество (2)

## Ожидаемый результат
1. При повторном нажатии на кнопку добавления товара с страницы "Товары" количество этого товара в корзине увеличивается
2. Отображаемое количество товара синхронизировано между страницей "Товары" и корзиной
