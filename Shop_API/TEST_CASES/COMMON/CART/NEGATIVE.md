# CART_NEG_001

## Название
Товар со статусом "Нет в наличии" недоступен для добавления в корзину

## Связанные требования
1. REQ_CART_001 — Добавление товара в корзину (п. 1.3)

## Предусловия
1. В системе существует пользователь t_user
2. Пользователь t_user авторизован и находится в разделе "Товары"
3. В системе существует хотя бы один товар со статусом "Нет в наличии"

## Шаги воспроизведения
1. Найти товар со статусом "Нет в наличии"

## Ожидаемый результат
1. Напротив товара со статусом "Нет в наличии" не отображается иконка добавления в корзину
2. Возможность добавить такой товар в корзину отсутствует

# CART_NEG_002

## Название
Товар в количестве 51 единицы недоступен для добавления в корзину

## Связанные требования
1. REQ_CART_001 — Добавление товара в корзину (п. 1.8)

## Предусловия
1. В системе существует пользователь `t_user`
2. Пользователь `t_user` авторизован и находится в разделе "Товары"
3. В системе существует хотя бы один товар со статусом "В наличии"
4. Корзина пользователя `t_user` пуста

## Шаги воспроизведения
1. Нажать кнопку добавления в корзину напротив товара со статусом "В наличии" 51 раз
2. Нажать на иконку корзины в интерфейсе для просмотра содержимого корзины

## Ожидаемый результат
1. Товар не добавляется в корзину в количестве больше 50
2. Отображается ошибка: `"Максимальное количество одной позиции - 50"`
3. У товара в корзине количество не превышает 50
4. На круглой кнопке с иконкой корзины напротив того же товара количество не превышает 50

# CART_NEG_003

## Название
Превышение максимальной суммы корзины недоступно

## Связанные требования
1. REQ_CART_004 — Расчёт итоговой стоимости корзины (п. 4.9)

## Предусловия
1. В системе существует пользователь `t_user`
2. Пользователь `t_user` авторизован и находится в разделе "Товары"
3. В системе существуют 3 товара со статусом "В наличии":
   - товар 1 с ценой `1 000 000`;
   - товар 2 с ценой `1 000 000`;
   - товар 3 с ценой `1`
4. Корзина пользователя `t_user` пуста

## Шаги воспроизведения
1. Добавить в корзину товар 1 в количестве `50` штук
2. Добавить в корзину товар 2 в количестве `50` штук
3. Добавить в корзину товар 3 в количестве `1` штука
4. Открыть корзину.

## Ожидаемый результат
1. При попытке добавить товар 3 сумма корзины не должна превышать `100 000 000`
2. Добавление товара 3 блокируется
3. Общая сумма корзины не превышает `100 000 000`
4. Количество товаров и суммы в корзине остаются в пределах максимального лимита

# CART_NEG_004

## Название
Отказ в оплате корзины при недостаточном балансе кошелька

## Связанные требования
1. REQ_CART_006 — Оформление и оплата корзины (п. 6.3)

## Предусловия
1. В системе существует пользователь `t_user`
2. Пользователь `t_user` авторизован и находится в разделе "Товары"
3. В корзине пользователя находится хотя бы один товар
4. В кошельке пользователя находится недостаточная сумма для оплаты корзины

## Шаги воспроизведения
1. Перейти в корзину

1. Операция оплаты не выполняется
2. Корзина и её содержимое остаются без изменений
3. Пользователю отображается сообщение об ошибке: `Недостаточно средств на балансе`
