# WALLET_POS_001

## Название
Успешное отображение баланса кошелька

## Связанные требования
1. REQ_WALLET_001 - Кошелёк пользователя (п. 2.1.1)

## Предусловия
1. В системе существует авторизованный пользователь t_user

## Шаги воспроизведения
1. Перейти в раздел "Кошелёк"
2. Проверить отображение баланса

## Ожидаемый результат
1. Баланс отображается в формате `999 999.99`— с пробелами-разделителями тысяч и запятой в качестве десятичного разделителя.

# WALLET_POS_004

## Название
Успешная фильтрация операций кошелька по типу "Пополнение"

## Связанные требования
1. REQ_WALLET_001 - Кошелёк пользователя (п. 2.2.2)

## Предусловия
1. В системе существует авторизованный пользователь t_user
2. Существует хотя бы одна операция с типом "Пополнение"
3. Существует хотя бы одна операция с типом "Списание"

## Шаги воспроизведения
1. Перейти в раздел "Кошелёк"
2. В выпадающем списке "Тип операции" выбрать значение "Пополнение"

## Ожидаемый результат
1. Отображается список операций только с типом "Пополнение"
2. Операции других типов (например, "Списание") не отображаются
