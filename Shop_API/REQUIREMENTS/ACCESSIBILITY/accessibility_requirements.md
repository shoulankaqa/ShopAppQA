# REQ_ACCESSIBILITY_001 — Требования к доступности

## Навигация с клавиатуры

1.1. Пользователь должен иметь возможность использовать клавиатуру для навигации по интерфейсу.  
1.2. Все интерактивные элементы страницы (ссылки, кнопки, поля ввода, чекбоксы, выпадающие списки и др.) должны быть доступны для навигации с клавишей Tab.  
1.3. Порядок фокуса должен соответствовать логическому порядку отображения элементов на странице.  
1.4. Фокус должен быть визуально заметен (подсветка, граница, изменение цвета) и не должен скрываться другими элементами.  
1.5. Пользователь не должен сталкиваться с focus traps, кроме специально предусмотренных случаев (модальные окна, выпадающие списки).

## Модальные окна

1.6. При открытии модального окна фокус клавиатуры должен сразу переходить на первый интерактивный элемент окна.  
1.7. Пока модальное окно открыто, фокус клавиатуры должен быть ограничен только элементами этого окна (focus trap).  
1.8. При закрытии модального окна фокус клавиатуры должен возвращаться на элемент, с которого окно было открыто.  
1.9. Пользователь должен иметь возможность закрыть модальное окно с помощью клавиши Escape.  
1.10. Модальное окно должно содержать чёткий заголовок, идентифицирующий его содержание.

## Выпадающие списки

1.11. Пользователь должен иметь возможность открыть выпадающий список с помощью клавиши Tab и нажать Enter/Space для выбора элемента.  
1.12. При открытом выпадающем списке фокус должен быть ограничен элементами списка.  
1.13. Пользователь должен иметь возможность закрыть выпадающий список с помощью клавиши Escape.  
1.14. При навигации внутри списка фокус должен автоматически переходить на соответствующий элемент при использовании клавиш ↑/↓.  
1.15. Выбранный элемент должен быть визуально отмечен (выделен цветом).
