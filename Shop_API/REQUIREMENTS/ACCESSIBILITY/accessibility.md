# REQ_ACCESSIBILITY_001 - Требования к доступности

## Навигация с клавиатуры

1. Пользователь должен иметь возможность использовать клавиатуру для навигации по интерфейсу.
2. Все интерактивные элементы страницы должны быть доступны для навигации с клавишей Tab.
3. Фокус должен быть визуально заметен (подсветка, граница, цвет).

## Модальные окна

1. При открытии модального окна фокус клавиатуры должен сразу переходить на первый интерактивный элемент окна.
2. Пока модальное окно открыто, фокус клавиатуры должен быть ограничен только элементами этого окна (focus trap).
3. При закрытии модального окна фокус клавиатуры должен возвращаться на элемент, с которого окно было открыто.
4. Пользователь должен иметь возможность закрыть модальное окно с помощью клавиши Escape.

## Выпадающие списки

1. Пользователь должен иметь возможность открыть выпадающий список с помощью клавиши Tab и нажать Enter/Space для выбора элемента.
2. При открытом выпадающем списке фокус должен быть ограничен элементами списка.
3. Пользователь должен иметь возможность закрыть выпадающий список с помощью клавиши Escape.
