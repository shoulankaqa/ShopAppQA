## REQ_AUTH_004 — Авторизация пользователя

1. Пользователь должен иметь возможность авторизоваться с использованием валидных учетных данных.
2. При вводе неверных учетных данных система должна отображать сообщение: "Неверные учетные данные".
3. После успешной авторизации пользователь должен быть перенаправлен в раздел "Товары".
4. После успешной авторизации система должна отображать сообщение: "Успех. Авторизовались под: <имя пользователя>".

---

## REQ_AUTH_005 — Выход из системы 

1. Пользователь должен иметь возможность выйти из системы.
2. После выхода пользователь должен быть перенаправлен на страницу авторизации.
3. После выхода пользователь не должен иметь доступ к защищенным разделам системы без повторной авторизации.
