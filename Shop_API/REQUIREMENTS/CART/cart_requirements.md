## REQ_CART_001 — Добавление товара в корзину

1. Пользователь должен иметь возможность добавить товар в корзину.  
2. После добавления корзина должна обновляться.  
3. Добавить товар можно:
   - из каталога товаров, по нажатию круглой кнопки с иконкой корзины напротив товара
4. При добавлении товара с количеством больше 1 в корзину добавляется указанное количество.  
5. Если товар уже есть в корзине, при повторном добавлении количество товара в корзине увеличивается.

---

## REQ_CART_002 — Удаление товара из корзины

1. Пользователь должен иметь возможность удалить товар из корзины.  
2. Итоговая стоимость должна пересчитываться после удаления товара.  
3. Удаление можно выполнить:
   - по кнопке "Убрать позицию" для отдельного товара;
   - по кнопке "Очистить корзину" (удаление всех товаров из корзины).  
4. После удаления товар исчезает из списка товаров в корзине.

---

## REQ_CART_003 — Изменение количества товара

1. Пользователь должен иметь возможность изменять количество товара в корзине.  
2. Количество товара не может быть меньше 1.  
3. При изменении количества товара стоимость позиции должна пересчитываться автоматически.  
4. Изменения должны отображаться пользователю без перезагрузки страницы.    
5. При увеличении количества стоимость позиции увеличивается, при уменьшении — уменьшается.

---

## REQ_CART_004 — Расчёт итоговой стоимости корзины

1. Система должна рассчитывать итоговую стоимость товаров в корзине.  
2. Итоговая стоимость должна изменяться при:
   - добавлении товара
   - удалении товара
   - изменении количества товара  
3. Итоговая стоимость должна отображаться пользователю в корзине.  
4. Стоимость каждого товара должна учитываться при расчёте общей суммы (цена товара × количество).  
5. Итоговая стоимость корзины равна сумме стоимостей всех позиций.  
6. Если в корзине нет товаров, отображается сообщение "Корзина пуста", итоговая стоимость не отображается.

---

## REQ_CART_005 — Просмотр корзины

1. Пользователь должен иметь возможность просматривать содержимое корзины.  
2. В корзине для каждого товара отображаются:
   - наименование товара
   - цена товара
   - выбранное количество 
3. В корзине отображается итоговая стоимость всех товаров.  
4. Если в корзине нет товаров, отображается сообщение о пустой корзине (например, "Корзина пуста").
