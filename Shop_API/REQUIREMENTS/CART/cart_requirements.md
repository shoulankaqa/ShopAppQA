## REQ_CART_001 — Добавление товара в корзину

1. Пользователь должен иметь возможность добавить товар в корзину.
2. После добавления корзина должна обновляться.

---

## REQ_CART_002 — Удаление товара из корзины

1. Пользователь должен иметь возможность удалить товар из корзины.
2. Итоговая стоимость должна пересчитываться.

---

## REQ_CART_003 — Изменение количества товара 

1. Пользователь должен иметь возможность изменять количество товара.
2. Количество товара не может быть меньше 1.
3. При изменении количества товара стоимость позиции должна пересчитываться автоматически.
4. Изменения должны отображаться пользователю без перезагрузки страницы.

---

## REQ_CART_004 — Расчет итоговой стоимости корзины

1. Система должна рассчитывать итоговую стоимость товаров в корзине.
2. Итоговая стоимость должна изменяться при:
   - добавлении товара
   - удалении товара
   - изменении количества товара.
3. Итоговая стоимость должна отображаться пользователю в корзине.
4. Стоимость каждого товара должна учитываться при расчете общей суммы.

---

## REQ_CART_005 — Просмотр корзины
1. Пользователь должен иметь возможность просматривать содержимое корзины.
