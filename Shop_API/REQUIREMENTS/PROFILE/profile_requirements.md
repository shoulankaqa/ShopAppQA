## REQ_PROFILE_001 — Общие требования к профилю

1. Пользователь должен иметь возможность изменить данные профиля после авторизации.
2. Изменения должны сохраняться после подтверждения.

---

## REQ_PROFILE_002 — Управление email

1. Пользователь может добавить или изменить email в профиле.
2. Email используется для восстановления пароля.
3. Email должен соответствовать формату example@domain.com.
4. При изменении email система сохраняет новое значение.

 ---

## REQ_PROFILE_003 — Изменениe пароля

1. Пользователь должен иметь возможность изменить пароль.
2. Для изменения пароля необходимо ввести текущий пароль.
3. Новый пароль должен соответствовать требованиям REQ_AUTH_002 — Требования к паролю.
