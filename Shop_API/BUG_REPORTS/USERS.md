# BUG_USER_DELETE_001

## Заголовок
500 ошибка при удалении пользователя с историей пополнения кошелька

## Описание
При попытке удалить пользователя, ранее выполнявшего пополнение кошелька, система возвращает 500 Internal Server Error, и пользователь не удаляется из системы.

## Шаги воспроизведения
1. Авторизоваться под учетной записью tadmin
2. Перейти в раздел "Пользователи"
3. Выбрать пользователя с историей пополнения кошелька (например, anka)
4. Нажать на кнопку "Удалить"

## Ожидаемый результат
1. Пользователь успешно удаляется из системы.

## Фактический результат
1. Отобразилось 500 Internal Server Error
2. Пользователь не удалился

Backend log:
QueryFailedError: update or delete on table "users" violates foreign key constraint on table "wallet".
