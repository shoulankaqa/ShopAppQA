# BUG_COMMON_001 (Исправлен)

## Название
Фокус клавиатуры (Tab) не переходит сразу в модальные окна системы

## Описание
При открытии модальных окон (например, удаление товара или пользователя):
 - фокус не переходит сразу в окно
 - Tab продолжает идти по основной странице
 - фокус не ограничен окном

## Предусловия
2. Пользователь авторизован под логином tadmin (admin)
3. В системе существуют:
   - хотя бы один товар
   - хотя бы один пользователь

## Шаги воспроизведения
1. Открыть раздел "Товары"
2. Нажать кнопку "Удалить" у любого товара (открывается модальное окно подтверждения удаления товара)
3. Нажимать клавишу Tab и наблюдать, куда переходит фокус
4. Закрыть модальное окно
5. Открыть раздел "Пользователи"
6. Нажать кнопку "Удалить" у любого пользователя (открывается модальное окно подтверждения удаления пользователя)
7. Нажимать клавишу Tab и наблюдать, куда переходит фокус

## Ожидаемый результат
1. При открытии модального окна фокус сразу переходит на первый интерактивный элемент окна (например, кнопку "Удалить" или "Отмена")
2. Пока модальное окно открыто, фокус ограничен только его элементами:
   - цикл Tab/Shift+Tab перемещает фокус по элементам внутри окна
   - фокус не выходит на элементы основной страницы
3. После закрытия модального окна фокус возвращается на элемент, с которого это окно было открыто

## Фактический результат
1. При открытии модального окна фокус не переходит сразу на элементы этого окна
2. При нажатии Tab фокус переходит по элементам основной страницы за модальным окном
3. Фокус не ограничен рамками модального окна: можно продолжать обходить Tab-ом элементы страницы, пока модальное окно остаётся открытым
