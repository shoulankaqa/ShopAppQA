# Общие тестовые данные (только для локального стенда)

> ВНИМАНИЕ: это тестовые пользователи для локальной разработки и учебных целей.
> Эти логины и пароли не используются в продакшене.

## Пользователи

| ID         | Роль                   | Логин      | Пароль      | Где использовать |
|------------|------------------------|------------|------------|------------------|
| U_ADMIN    | Админ                  | `tadmin`    | `Admin123!` | Тесты прав доступа, админские сценарии |
| U_SELLER   | Селлер                 | `tseller`   | `Seller123!` | Тесты прав доступа селлера |
| U_USER     | Обычный пользователь   | `tuser`     | `User123!`  | Базовые позитивные и часть негативных тестов для обычного юзера |
| U_EXIST    | Уже существующий юзер   | `existuser` | `Exist123!`  | Тесты, где важен факт, что логин уже зарегистрирован (POS/NEG) |
| U_FAKE     | Несуществующий юзер     | `fakeuser`    | `Fake123!`  | Негативные тесты: пользователь отсутствует в системе |
