# QA E-Commerce Project

Тестовая документация для проекта интернет-магазина.

---

# Project Overview

Проект включает:
- функциональное тестирование
- нефункциональное тестирование
- API тестирование
- тест-кейсы
- чек-листы
- баг-репорты
- требования к системе

---

# Scope

## Authentication
- регистрация
- авторизация
- выход с аккаунта
- восстановление пароля

## Products
- просмотр товаров
- создание товара
- редактирование товара
- удаление товара
- фильтрация

## Cart
- добавление товара
- удаление товара
- изменение количества

## Orders
- оформление заказа

## Users (ADMIN)
- создание пользователей
- изменение пользователей
- удаление пользователей
- управление ролями

## Settings
- изменение профиля
- изменение пароля
- настройка темы интерфейса
